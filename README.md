# PLAEE EXERCISE

This app is a basic version of the Polymarket site. Currently only has the main page, containing some event cards pulled form polymarket using their API. The site was created using React and Vite.

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

To run, simply 

```js
npm install
npm run dev
```
